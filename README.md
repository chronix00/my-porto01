Ini adalah repositori untuk website portofolio profesional saya, **FransDev**. Situs ini dirancang untuk menawarkan jasa pembuatan website bagi UMKM maupun individu dengan fokus pada kemudahan, kualitas standar industri, dan harga yang bersahabat.

## 🔗 Live Demo
Lihat web di sini: [https://chronix00.github.io/my-porto01/](https://chronix00.github.io/my-porto01/)

## 🚀 Fitur Utama
- **Landing Page Interaktif:** Tampilan modern dan responsif (nyaman dibuka di HP maupun desktop).
- **Katalog Layanan Spesifik:** Penjelasan detail layanan dari sisi teknis hingga fungsional.
- **Formulir Pemesanan Langsung:** Integrasi konsep pemesanan yang memudahkan klien memilih tema dan mengirim detail via WhatsApp.
- **Informasi Harga Transparan:** Membantu calon klien menentukan paket sesuai budget mereka.
